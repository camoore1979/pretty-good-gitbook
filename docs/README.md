This is an example of using gitbook. Gitbook is really cool.

## Running pretty-good-gitbook

### clone this repo

    $ git clone https://github.com/camoore1979/pretty-good-gitbook.git

### install gitbook-cli

    $ npm install -g gitbook-cli

### install the plugins

    $ npm run docs-install
    
### build the gitbook

    $ npm run docs-build
    
### serve the gitbook

    $ npm run docs-serve
